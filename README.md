# Calculadora em Java

# Sobre o projeto

Calculadora desenvolvida em interface Swing em Java com propriedades básicas: somar, subtrair dividir e multiplicar. 
Foram adicionados as funcionalidades de limpar, voltar e on/of dos botões.
